## Cómo ejecutar proyecto:

```sh
$ cd mipergamino
```

```sh
$ npm update
```

```sh
$ npm install
```

```sh
$ npm run dev
```

Renombrar `.env.sample` a `.env` y modificar las variables `MONGO_USERNAME` y `MONGO_PASSWORD` (solicitar).
